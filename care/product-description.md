# Consistent Product Descriptions

## Framework: CARE

## Fields

**Context:** We need to write product descriptions for Promptary across multiple platforms — directories, app stores, social profiles, and partner pages. Each platform has different length limits but the description must be recognizably the same product.

**Action:** Write a product description for Promptary that can be adapted to different lengths.

**Result:** A master description (100 words), a short version (50 words), and a micro version (25 words). All three should use the same core framing and key phrases so they reinforce each other across platforms.

**Example:** Langfuse — Open-source LLM engineering platform. Debug, analyze, and iterate on your LLM app. Integrates with any model and framework. Used by teams building reliable AI products. (Short, specific, leads with what it does, ends with who uses it.)
