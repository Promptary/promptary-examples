# In-App Onboarding Tooltip Copy

## Framework: CARE

## Fields

**Context:** Promptary's canvas app has several features that new users don't discover on their own: the right-click context menu to edit prompts, the middle-click to delete, and the API key section in the prompt editor. These features need tooltip or onboarding hint copy.

**Action:** Write the copy for 3 onboarding hints — one for each of those three features.

**Result:** Each hint: one sentence explaining what the feature does, one sentence on why it's useful. Under 30 words per hint. Plain language, no jargon.

**Example:** Linear's onboarding hints — "Press G then I to go to your inbox. Triage issues without leaving the keyboard." Short, specific, teaches the behavior and the benefit in one breath.
