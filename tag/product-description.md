# AI Tool Directory Product Description

## Framework: TAG

## Fields

**Task:** A product listing for Promptary on an AI tool directory. The listing allows 150 characters for a short description and 500 characters for a long description.

**Action:** Write both the short description (under 150 characters) and the long description (under 500 characters) for the directory listing.

**Goal:** A developer or technical founder browsing the directory should immediately understand what Promptary does, who it's for, and what makes it different from general prompt libraries — and click through to learn more.
