# Blog Post Illustration — REST API Concept

## Framework: Image Gen

## Fields

**Subject:** An abstract visual metaphor for a REST API endpoint — glowing data packets traveling along clean geometric lines between three floating nodes. Left node labeled "Promptary", middle is the API path, right node is "Your App". The nodes pulse with soft light.

**Style:** Flat vector illustration with subtle depth and glow, tech blog aesthetic, similar to Stripe or Tailwind documentation illustrations. Not photorealistic. Clean, modern, slightly playful.

**Lighting:** Very dark background (#09090F), vibrant teal (#5EEAD4) and deep purple (#7B34FE) elements, no harsh shadows, subtle inner glow on lines.

**Composition:** Wide horizontal (16:9), three nodes spread across the frame with clear left-to-right visual flow representing a request-response cycle. Centered vertically with breathing room.

**Mood:** Clear, technical, approachable. Makes a complex concept feel simple and understandable. The feeling of "oh, I get it now."

**Negative:** People, faces, photorealism, cluttered background, text labels, watermark, stock photo aesthetic, neon cyberpunk excess.

**Parameters:** --ar 16:9 --style raw --v 6.1 --no text
