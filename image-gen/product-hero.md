# Product Hero Image — Dark UI Aesthetic

## Framework: Image Gen

## Fields

**Subject:** A minimalist dark-themed web application interface showing glowing teal bubble nodes connected by thin lines, floating in deep space. The bubbles are labeled with abstract text representing AI prompts organized into topics.

**Style:** Cinematic 3D render, dark UI aesthetic, similar to Linear or Vercel product marketing visuals. Clean, premium, technical.

**Lighting:** Deep space background (#09090F dark), neon teal accent glow (#5EEAD4) emanating from the bubble nodes, subtle purple ambient glow, soft lens flare on the brightest nodes.

**Composition:** Slightly angled (15 degrees), interface filling 70% of frame, floating above a dark gradient. Wide horizontal format. Elements arranged with visual hierarchy — larger topic bubbles containing smaller prompt bubbles.

**Mood:** Futuristic, premium, calm. The feeling of powerful software that is also beautiful to use. Not dystopian — quietly impressive.

**Negative:** Text, watermark, blurry, low resolution, cluttered background, people, faces, UI chrome like window frames or scrollbars, gradients that look like stock photos.

**Parameters:** --ar 16:9 --v 6.1 --style raw --q 2
