# Canvas Feature Screenshot — Marketing Use

## Framework: Image Gen

## Fields

**Subject:** A screenshot-style render of the Promptary physics canvas showing 4 colored topic bubbles (teal, purple, orange, green) each containing 3-5 smaller prompt bubbles orbiting them. The bubbles have subtle rim lighting and float in dark space. One bubble is highlighted as if being hovered.

**Style:** UI screenshot aesthetic — realistic enough to look like an actual product but slightly idealized. Similar to how Notion or Figma show their product in marketing materials.

**Lighting:** Dark background, each topic bubble glowing with its own color, the hovered bubble has a brighter rim glow, subtle ambient light from all bubbles combined.

**Composition:** Fill the entire frame with the canvas — no window chrome or browser frame. Slight zoom effect as if the camera is slowly pushing in. 16:9.

**Mood:** Alive, organized, visual. The opposite of a list of prompts in a spreadsheet. Should make the viewer think "I want to use this."

**Negative:** Browser chrome, window frames, scrollbars, cursor, watermark, text that is readable (abstract blurred text is fine), logos.

**Parameters:** --ar 16:9 --v 6.1 --style raw
