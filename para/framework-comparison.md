# Technical Comparison — RACE vs RISEN vs CO-STAR

## Framework: PARA

## Fields

**Problem:** Developers and content teams new to prompt engineering face a confusing landscape of frameworks with similar-sounding names. They need to understand which framework to use for which task without having to try all of them through trial and error.

**Approach:** Compare RACE (Role, Action, Context, Expectation), RISEN (Role, Instructions, Steps, End Goal, Narrowing), and CO-STAR (Context, Objective, Style, Tone, Audience, Response) across three dimensions: what they optimize for, when to use them, and when NOT to use them.

**Result:** An 800-word comparison article with a summary table at the top (framework, best for, fields, difficulty), followed by a dedicated section for each framework explaining its strengths and ideal use cases, ending with a decision flowchart in text form ("If your task is X, use Y because Z").

**Application:** Developers and non-technical users who have heard about prompt frameworks and want a definitive guide to choosing between the three most popular ones. They will save and share this article — it needs to be the resource they bookmark.

**Rules:** The summary table must come first, before any prose. Each framework section must include one concrete example prompt using that framework. The decision flowchart must cover at least 5 different task types. Do not recommend one framework as universally superior.
