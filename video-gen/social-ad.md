# Social Media Ad — 6 Second Hook

## Framework: Video Gen

## Fields

**Scene:** A developer's dark terminal screen with a single curl command typed out. The command hits Enter. A JSON response appears with a "text" field containing a complete AI prompt. Cut to the same prompt appearing in a chatbot response that looks polished and professional.

**Motion:** Text types character by character (terminal style). After Enter, a loading indicator flashes briefly, then the JSON response appears line by line. Cut is hard but not jarring.

**Camera:** Static close-up on the terminal. No camera movement — the motion is all in the content.

**Lighting:** Dark background, terminal green text on dark terminal background for the first half, then transitions to the app UI with teal/purple color scheme.

**Style:** Realistic terminal aesthetic first half, clean modern app UI second half. Documentary style — looks like a real developer doing real work.

**Duration:** 6 seconds

**Negative:** Stock footage aesthetic, voiceover, subtitles, watermark, faces, people typing (hands only if needed), generic laptop shots.

**Parameters:** 1080p, 24fps, motion_strength: 0.4 (text motion should be controlled)
