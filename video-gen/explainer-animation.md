# Explainer Animation — Prompt API Concept

## Framework: Video Gen

## Fields

**Scene:** Abstract dark space with three glowing geometric nodes connected by luminous lines. Left node pulses with teal light labeled "Prompt" (abstract, not readable). A packet of light travels along the line to a middle node labeled "API". Another packet travels to the right node labeled "App". The cycle repeats.

**Motion:** Light packets travel smoothly from left to right. Each node pulses when a packet arrives. After two cycles, zoom out to reveal multiple apps on the right receiving the same light from the same left node.

**Camera:** Static wide shot for the first 4 seconds. Then slow zoom out over 2 seconds to reveal the broader system. Ends on the wide view.

**Lighting:** Black background, teal (#5EEAD4) for the Prompt node and its line, purple (#7B34FE) for the API node, warm white for the App nodes. Subtle glow halos around each node.

**Style:** Motion graphics / kinetic design. Clean, minimal, geometric. Similar to explainer animations from Stripe or Cloudflare.

**Duration:** 6 seconds, loopable

**Negative:** Text that is readable, photorealism, people, faces, cluttered elements, jarring cuts, flickering.

**Parameters:** 1080p, 30fps, motion_strength: 0.6
